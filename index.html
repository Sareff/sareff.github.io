Создание Веб-сайта

Интернет

это крупнейшая в мире библиотека, в которой все книги разбросаны по полу.

Юзабилити

Качественный показатель простоты и удобства использования сайта. В зависимости от вида сайта правила юзабилити будут адаптироваться по-своему. Юзабилити сайта оценивается по 5 компонентам качества: ориенация, эффективность, запоминаемость, удовлетворенность, ошибки.

Дизайн

Внешний вид сайта, кнопки, шапка, меню, блоки и т.д. Это тот визуальный образ, который видит пользователь сайта. Внешний вид сайта это, пожалуй, то что и делает интернет таким неподражаемым. Основная задача любого дизайна сайта – это объединение всех блоков с информацией в одну лаконичную и красивую систему.

Разработка

Создание структуры HTML-кода, размещающего элементы веб-страницы (текст, изображение, мультимедиа и т.д.) в окне браузера, согласно в заранее подготовленному макету, чтобы элементы на сайте располагались так же, как и на макете. Существует множество вариантов верстки сайтов.

Программная часть

Любой сайт на просторах Интернета не обходится без программной части, будь то одностраничник или большой интернет-магазин. Благодаря этим инструментам интернет стал таким какой он есть на данный момент.

JavaScript

Язык программирования, который добавляет интерактивности веб-сайту.

CSS

Формальный язык описания внешнего вида веб-сайта, написанного на HTML.

HTML

Стандартизированный язык разметки документов во Всемирной паутине. Большинство веб-страниц содержат описание разметки на языке HTML.

PHP

Скриптовый язык, применяемый для разработки веб-приложений.

SQL

Язык программирования, применяемый для управления базами данных.

Программное обеспечение для разработки веб-сайтов

На данный момент все сайты разрабатываются людьми, по этому помимо инструментов разработки им нужны также среды разработки, где они смогут всеми этими инструментами воспользоваться.

Sublime Text 3. Довольно быстрый и удобный текстовый редактор с подсветкой кода, который так же поддерживает всякого рода пакеты и плагины.

Atom. Бесплатный текстовый редактор с открытым исходным кодом для Linux, macOS, Windows с поддержкой плагинов, написанных на Node.js, и встраиваемых под управлением Git. Большинство плагинов имеют статус свободного программного обеспечения, разрабатываются и поддерживаются сообществом.

JetBrains WebStorm. Интегрированная среда разработки на JavaScript, CSS & HTML от компании JetBrains, разработанная на основе платформы IntelliJ IDEA.WebStorm обеспечивает автодополнение, анализ кода на лету, навигацию по коду, рефакторинг, отладку, и интеграцию с системами управления версиями.

Dreamweaver. Визуальный HTML-редактор компании Adobe. Программа также имеет подсветку синтаксиса и поддержку всех интернет-иструментов разработки сайтов. Основная функциональность программы сосредоточена на создании HTML и CSS кода, а также работе с файлами сайта
